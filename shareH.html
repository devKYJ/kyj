가계부 : Cash Book

소개

Visual Studio 윈폼, 텍스트파일 읽기쓰기를 이용한 가계부를 구현해보았습니다.

Project information

  • Title: Cash Book
  • Category: C#
  • personnel: 개인

주요기능

  • 로그인 : 이름, 비밀번호 직접 지정하여 로그인
  • 입금, 출금: 텍스파일 읽기 쓰기 사용
  • 월변경: 기록된 달 중 원하는 달로 변경 가능 (불러오기 기능)
  • 월마감: 현재 선택된 달의 급여를 다음달 1일에 데이터 기록

Cash Book 실행화면

프로그램 실행시 화면입니다. 제가 지정한 아이디와 비밀번호만 로그인이 가능합니다. 로그인 후 저장된 파일을 불러옵니다. (로그인 하지않을 경우 입금,출금 비활성화 됨)


 

 

입금 선택 시 화면

 

내역 입력후 합산된 잔액 화면

 

출금 선택 시 화면

 

내역 삭제 화면
삭제할 내역을 선택 후 메세지로 띄워줍니다. (내역 더블클릭시 내역 편집 가능)

 

삭제 된 내역 확인 화면

 

월마감 선택시 화면
현재 선택된 달의 잔액데이터를 다음달 1일 입금일에 기록되는 기능

 

월마감 후 이월된 데이터 화면

 

월변경 화면
변경할 월 리스트를 띄워줍니다. 선택한 해당 월 내역으로 변경됩니다.
